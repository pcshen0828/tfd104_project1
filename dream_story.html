從愛吃甜點的女孩到成立第一間烘焙工作室：
Canelé Studio 創辦人 Linda Wang
posted on 2021-10-01 by set up dream developer
▲ 圖片是位於台北城西的 Canelé Studio 一景，店名源自法國甜點「可麗露」，是 Linda 最喜愛的一種甜點，也是烘焙工作室的招牌品項。
和 Linda 相約在 Canelé Studio 訪談的那天下午，是夏末秋初、陽光飽滿但帶點微涼的九月天氣。五年前的同一個季節，Canelé Studio 在台北城西的街角誕生。在周圍帶有古色的建築群落簇擁下，Canelé Studio 典雅而內斂的性格毫不突兀地鑲嵌其中，卻又以一種優雅自信的姿態，悄然宣示著自己的存在。
「嗨！你們一定就是 set up 團隊吧，快進來呀！」一推開 Canelé Studio 工作室的門，便聽見 Linda 溫暖而爽朗的招呼聲。不大的坪數中，麵包與甜點展示架、吧檯、內用木長桌和中島工作區，井然有序地依動線羅列著。為了店內新的季節限定甜點，Linda 正與夥伴們圍繞著中島進行討論。「正好你們來了，可以給我們一點建議」，Linda 和幾個夥伴端出一排閃耀著秋天大地色澤的精緻甜點，開心與我們分享。而在五年前，這種與團隊一起討論的熱絡場景，對 Linda 來說是一種意想不到的奢侈。
▲ Canelé Studio 今年秋天推出的季節限定甜點套餐：「Avant la neige 降雪之前」。左側以胡桃、榛果及焦糖為基底製成的堅果塔，上方用蛋白霜推疊出螺旋霜體，微烤至焦脆，營造清爽不膩的口感；右側則是以杏仁粉烤製而成的濕潤糕體，表面塗上以法國的杏桃、台灣的金棗和蜂蜜熬煮成的果醬，浸漬一晚，呈現金黃光澤。後方的提拉米蘇，特別選用日本焙茶提粹而成的糖液取代傳統的咖啡酒，搭配馬斯卡彭起司，捎來一股秋日颯爽的禪意。
最初的想法：打造一個能夠帶給人溫暖的場所
那時，她從法國攻讀珠寶設計學成歸國，在相關領域工作約二年，烘焙是休閒時的興趣，也有在 Instagram 上經營自己的甜點帳號，偶爾與親朋好友們分享。儘管心中懷有開一家店的夢想，卻從來沒想過要以此為正業。對 Linda 來說，烘焙真正迷人的地方在於：食材透過美感與美味的設計，讓人看到時自然嶄露發自內心的微笑與讚嘆；以甜點為媒介，串起了人與人之間相處的溫度。正是這樣的想法，讓 Linda 開始認真思考自己或許可以經營一個空間，讓人可以在這裡享受甜點帶來的五感饗宴，自己作為空間主人，不僅做著自己真正喜愛的事，更將這份溫度透過美味的食物傳遞給每位來訪的客人。
夢想開始在 Linda 心中著根萌芽，越長越大，藍圖也越來越清晰，她開始思考要如何踏出第一步。「那時候整家店的藍圖只有我一個人知道，它就在我的腦袋裡，這家店就是我，我就是它。但是我知道，如果要做出我心目中那樣子有溫度的店，不能只有我自己做，會疲乏，疲乏就沒辦法提供好的服務。所以，我必須找到團隊。」原本就有在 set up community 夢想家聚落加入烘焙社團的 Linda，主動向有經驗的夥伴們請益，並著手開始畫下一張又一張的計畫圖：
▲ Canelé Studio 的第一張計畫草圖。
關鍵是「信念」：讓夢想在現實中茁壯
「我從小就算是有主見的小孩，對於自己想做的事情有明確的目標，加上後來學珠寶設計，要怎麼樣把腦袋中看見的東西畫下來、做出來，我覺得是很重要的，也是我一直在練習的。在真正開始做之前，其實我也不知道我有多大的能耐，可以走到哪裡。如果要問我實現夢想最重要的關鍵是什麼，我覺得是要對自己的夢想有信念。那種信念是說，我一定會看見它（夢想）從現實中長出來，而且我對它有責任，我要讓它長得很好。」
然而，興趣和真正經營一家店，畢竟是兩回事，這段孵化夢想的路，走得艱辛。Linda 在一開始的籌備階段，利用工作閒暇之餘進修學習，練習烘焙，目標是考取證照，一邊設計未來店裡的菜單。不僅如此，Linda 也積極參與經營管理相關主題的工作坊，累積商業知識，同時思考未來的營運模式，以及團隊分工的細節，「我不一定要做全部的事，但是一定要有 know-how，才能夠給予團隊他們需要的資源」，Linda 分享。
直到現在，Linda 和夥伴們固定會在 set up 上討論想法，激發創意；不僅如此，Canelé Studio 工作室也在夢想家聚落舉辦烘焙工作坊，將經驗分享給更多的人，「其實不只有我是給予的那個人，很神奇的是，我們在 set up 上遇見的同好朋友，往往帶給我們很大的力量和靈感，我想這就是有出、有進，這之中有什麼在發生。看著他們，我告訴自己，我也要更加努力。」
▲ Canelé Studio 每週六固定在店裡開設社區烘焙工作坊，讓附近的居民能夠自己手作麵包、糕點等，與家人朋友們分享，玩得開心、也吃得安心，重點是，增進彼此認識與交流的機會。店裡的常客，有來自美國移民搬來台灣的一家人，也是工作坊的學員。
用一個夢想，點亮另一個夢想
「我覺得對現在的人來說，quality time 是我們越來越重視卻也相對變得更難取得的東西。現代科技幫助我們跟人產生更快速的連結，但其中的質量、深度和溫度，是無法被量化的。那也是我想要創造的東西，你信任把你的時間交給我，而我能帶給你更有價值的互動、體驗和連結。」
這就是 Linda 在店裡開設社區烘焙工作坊的初衷。看著每天來 Canelé Studio 買麵包和糕點的客人，總是在上班前或下班後，行色匆匆的來去，可能是為了家人的溫飽，要買今天或明天的早餐，也可能是通勤來不及吃早餐的上班族，需要一個簡單能夠果腹的食物。「我常常覺得，食物能夠創造的，不只是生理上的飽足而已，還有一層是心理上的富足」，Linda 分享。於是 Linda 與夥伴們設計籌畫工作坊的內容和形式，思考如何創造帶得走的體驗，讓 Canelé Studio 不僅是一家店，更是串起人與人之間互動溫度的場所。
如今，工作坊已經進入第三年，有不少學員是家庭、伴侶、室友或是好朋友一起來參與，也有學員因為參與了工作坊，激發了烘焙的興趣，現在已經開始經營線上甜點餐車。「我想最珍貴的地方莫過於此吧！這些小亮點是我當初規劃夢想時沒有想過的。正因為有更多人的參與，他們帶我來到更遠的地方，他們延伸了我的夢想，帶走其中一小部分，去創造其他美好的事情。於是，我也可以乘著他們的翅膀，看見更遼闊的風景」，Linda 分享。
Canelé Studio 未來會走到哪裡？還很難說。我們只知道，這張夢想地圖會一直擴展下去，與更多人相遇。
