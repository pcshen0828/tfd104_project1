Hi! 夢想家
一起認識 set up
我們的使命
set up 是一款生產力工具，
致力於透過視覺化的方式，結合科技與人類心智的圖景，
協助使用者針對不同的目標，逐步建構自己的夢想實現藍圖，
分享並連結線上的使用者，形成支持性的社群系統。
視覺化你的想法
set up 相信每個人都是天生的夢想家，我們的大腦每天都有千千萬萬個新點子生成，無論是透過文字或圖像，展開你的第一步，首先用視覺化的方式記錄下這些想法。
建立專屬於你的地圖
將無形的想法轉換成可視的物件之後，第二步，思考如何透過適合的圖表（我該如何找到適合自己的圖表工具？）幫助我們梳理這些思緒，並開始建立執行步驟。
善用線上社群資源
一加一的力量無限大！在 set up community 隨時交流彼此的想法，你也可以主動撰寫訊息、尋求建議，隨時修正你的地圖。別忘了給予他人鼓勵與回饋！
圖表追蹤，隨時修正
夢想計畫的實現，是一連串修正與決策的動態過程。通過建立圖表的不同版本，檢視你的地圖可以如何修正？又有哪些不在計畫之內的美好風景？記錄下來。
